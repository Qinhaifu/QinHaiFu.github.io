1、IMPORT_NAME

2、IMPORT_FROM

然后是IMPORT_FROM，刚刚我们已经获取到了package，现在看一下IMPORT_FROM

这个case其实调用的就是import_from函数

这里的关键是x = PyObject_GetAttr(v, name);，它其实只是从v中获取了对象，这里的v就是刚才的package

3、结合源码解释现象

上面我们已经简单看了源码，现在我们回过头来看一下出现问题的代码，当使用from . import test2的时候，test2没有被正常的reload。
这是因为package中的test2一直存在，我们直接从package中获取test2，所以test2没有被reload。
